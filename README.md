Tours & Travels Web Application
Python | Django | SQLite | HTML5 | CSS3 | Bootstrap | PythonAnywhere | Django ORM | Authentication
Designed and developed a full-stack web application using Python and Django framework following MVT (Model-View-Template) architecture
Implemented complete CRUD operations to manage tour listings with fields including title, location, price, duration, and image
Built secure user authentication system including user registration, login, and session management using Django built-in authentication
Developed a tour booking system allowing users to browse and book available tour packages
Integrated image upload functionality for tour listings with dynamic data rendering on the frontend
Designed a fully responsive user interface using HTML5, CSS3, and Bootstrap with search functionality
Managed database operations using Django ORM with SQLite as the backend database
Successfully deployed the live application on PythonAnywhere in a production environment
